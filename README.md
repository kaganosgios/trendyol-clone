# Trendyol Filtre Sayfası Clone

Bu proje, **Next.js + Tailwind CSS** kullanılarak Trendyol’un mobilya listeleme/filtre ekranını görsel olarak klonlamak için hazırlanmış bir frontend çalışmasıdır.

## Özellikler
- Masaüstü ve mobil uyumlu arayüz
- Sol filtre alanında açılır/kapanır bloklar
- Checkbox ile filtre seçimi (UI state)
- Mobilde filtre/sıralama/kategori/marka/fiyat panelleri
- Ürün kart grid yapısı ve Trendyol benzeri sayfa düzeni

## Teknolojiler
- Next.js
- React
- Tailwind CSS
- TypeScript

## Kurulum
```bash
npm install
npm run dev
```

Tarayıcıda aç:
- [http://localhost:3000](http://localhost:3000)

## Not
Bu proje sadece frontend odaklıdır. Filtrelerin ve butonların işlevi backend'e bağlı değildir; amaç arayüzü görsel olarak eşleştirmektir.
